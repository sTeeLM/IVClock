# IVClock
## clock by stm32f103 and iv18, 使用IV18荧光管的时钟 ##

- 秒级别误差每年
- 温度显示
- 10个独立闹钟
- 万年历（从1901到2099）
- 语音整点报时报温度
- 闹钟铃声可定制
- 自带跑表和计时模式
- 蓝牙App远程控制和对时
- 节电模式，摇晃或者按键唤醒
- 根据光照智能调整IV18显示亮度
- 8.4V/5V外接电源、充电
